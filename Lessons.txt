Çoğul eki:  // Множественное число, суффикс
  a, ı, o, u - lar
  e, i, ö, ü - ler

Example:
  kitap - kitaplar
  defter - defterler
  çanta - çantalar
  kalem - kalemler

İstisna:
  saat - saatler
  seyahat - seyahatler
  hayal - heyaller

Isaret Zamirler:  // указательные местоимения
  Bu - yakın
  Şu - uzak
  O - çok uzak
  Bunlar - yakın
  Şunlar - uzak
  Onlar - çok uzak

Example:
  Bu ne? 
  Bu kalem
  Bu kalem değil
  Şu ne?
  Şu defter
  Şu defter değil

Kişi Zamirleri:
  Ben => um => muyum
  Sen => un => musun
  O => '' => mu
  Biz => uz => muyuz
  Siz => sunuz => musunuz
  Onlar => '' => mı

Example:
  Ayşe çalışkandır - O çalışkandır
  Ayşe, Fatma, ve Ali çalışkandır - Onlar çalışkandır

Zaman ve Zaman eki:  // Время и его суффикс
  Geçmiş zaman
  Şimdiki zaman:
    - Fiilin içinde a,ı => ıyor     // Ben anlatıyorum
    - Fiilin içinde e,i => iyor     // O yiyor
    - Fiilin içinde o,u => uyor     // O koşuyor
    - Fiilin içinde ö,ü => üyor     // O görüyor
    - Fiilin sonunda ı,i,u,ü => yor // okuyor,uyuyor
    - Fiilin sonunda a => ı,u       // bekliyor
    - Fiilin sonunda e => i,ü       // özlüyor
    Example:
      Ben geliyorum / Ben gidiyorum / Ben gitmiyorum / Ben gidiyor muyum? / Ben gitmiyor muyum?
      Sen geliyorsun / Sen gidiyorsun / Sen sevmiyorsun / Sen seviyor musun? / Sen sevmiyor musun?
      O geliyor / O gidiyor / O gelmiyor / O geliyor mu? / O gelmiyor mu?
      Biz geliyoruz / Biz gidiyoruz / Biz çalışmıyoruz / Biz çalışıyor muyuz? / Biz calışmıyor muyuz?
      Siz geliyorsunuz / Siz gidiyorsunuz / Siz sevmiyorsunuz / Siz seviyor musunuz? / Siz sevmiyor musunuz?
      Onlar geliyorlar / Onlar gidiyorlar / Onlar gelmiyorlar / Onlar geliyorlar mı? / Onlar gelmiyorlar mı?
  Gelecek zaman

Глухие: ç f h k p s ş t
Звонкие: b c d g ğ j l m n r v y z
p ç t к, при присоединении аффиксов, начинающихся на гласный, глухой согласный изменяется на соответствующий звонкий.
k изменяется в g только в том случае, если букве k предшествует буква n.
В корнях двусложных слов (т.е. тех, что состоят из 2х слогов), содержащих во втором слоге узкий гласный (ı, i, u, ü) при присоединении аффикса, начинающегося с гласной, происходит выпадение соответствующего узкого гласного. 
Аффикс утвердительной модальности -dır, -dir, -dur, -dür и -tır, -tir, -tur, -tür. Аффикс утвердительной модальности употребляется при определении предмета. Bыбор согласной t или d - закону гармонии согласных, при этом, если слово заканчивается на гласную, то автоматически следует аффикс, начинающийся с буквы d.

Ben   - ım im um üm 
Sen   - sın sin sun sün
O
Biz   - ız iz uz üz
Siz   - sınız siniz sunuz sünüz
Onlar - lar ler

Если слово оканчивается на гласный звук, а присоединяющийся аффикс начинается с гласного, то между двумя гласными используется вставочный Y.
Ben   - yım yim yum yüm 
Sen   - sın sin sun sün
O
Biz   - yız yiz yuz yüz
Siz   - sınız siniz sunuz sünüz
Onlar - lar ler
